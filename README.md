JavaScript |  Interface de linha de comando (CLI)

> Sistema de gerenciamento de estoque desenvolvido em Node.js, executado diretamente no terminal, com operações de CRUD (Criar, Ler, Atualizar e Deletar).

---

## Funcionalidades

- Listar todos os produtos do estoque  
- Adicionar novos produtos  
- Remover produtos existentes  
- Editar nome, valor ou quantidade  
- Buscar produtos por filtros
